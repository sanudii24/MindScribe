# MindScribe

Advanced AI-Powered Mental Health Assistant with Offline Capabilities

## About

MindScribe is a cutting-edge, privacy-first mental health companion powered by advanced AI technology. Designed to provide personalized mental health support with complete offline capabilities.

## Key Features

- 🔐 **Privacy-First**: Complete offline capability with zero data collection
- 🤖 **Advanced AI**: State-of-the-art LLMs with RAG memory architecture
- 🎙️ **Voice Support**: Natural voice interaction with TTS and speech recognition
- 💾 **Memory Intelligence**: Semantic memory system that learns and adapts
- 📱 **Cross-Platform**: Web, Desktop (Windows, macOS, Linux), and mobile
- ⚡ **High Performance**: Optimized inference engine with native CPU support

## Technology Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **AI/ML**: WebLLM, Transformers.js, ONNX Runtime, Whisper
- **Voice**: Piper TTS, Web Audio API, WASM
- **Desktop**: Tauri, Rust

## Getting Started

```bash
git clone https://github.com/sanudii24/MindScribe.git
cd MindScribe
npm install
npm run dev
```

## Building for Desktop

```bash
npm run tauri dev
npm run tauri build
```
